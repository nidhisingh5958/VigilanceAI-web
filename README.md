# VigilanceAI

The VigilanceAI project is structured to provide a modular and maintainable codebase for developing an AI-powered in-car wellness and emergency response system. Below is an overview of the project's architecture, including the main directories and files.

## Features
- Real-time wellness scoring
- Eye tracking (PERCLOS)
- Heart rate monitoring
- Emotion recognition
- Emergency protocol automation
- AI co-pilot assistance

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- Lucide Icons

## Installation
\`\`\`bash
npm install
npm run dev
\`\`\`

## Build for Production
\`\`\`bash
npm run build
\`\`\`
